README

This file provides an overview of the main functions performed by the data_cleaning.R script.
At the beginning of the script, the dataset is downloaded, unzipped, and loaded into R. The training and test datasets are then merged, assigned descriptive activity labels, and given clear and readable variable names. Finally, the script creates a tidy dataset and saves it as "tidy.txt".
Each stage of the process is clearly marked within the script using step labels, such as "Merge train and test sets (task 1)".
